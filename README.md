# Retail Sales Analysis

SQL, Python & Tableau analysis of Superstore sales data.

## Project Overview

- **Dataset**: Superstore Sales (9,995 transactions, 2014-2017)
- **Tools**: MySQL 8.0+, Python (Pandas), Tableau Public
- **Author**: Hoang Minh Nguyen

## Project Structure

retail-sales-analysis/
├── sql/           # SQL queries
├── python/        # Python scripts
├── tableau/       # Tableau dashboards
├── data/          # Datasets
├── docs/          # Documentation
├── LICENSE
└── README.md

## Analysis Goals

- Sales trends by Category, Region, Segment
- Profit analysis and loss identification
- Customer segmentation
- Window Functions and CTEs practice

## Tech Stack

- MySQL 8.0+ (Window Functions, CTEs)
- Python (Pandas, Matplotlib, Seaborn)
- Tableau Public
- Git & GitHub

## Getting Started

1. Clone this repo
2. Import Superstore CSV into MySQL
3. Run SQL queries from sql/ folder
4. Open Tableau dashboards from tableau/ folder
